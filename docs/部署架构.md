## 最新架构 

浏览器
  ↓
Next.js（公网） 香港云
  ↓（Server Actions / API Routes）
腾讯云nginx
  ↓
tv/dv（ZeroTier 内网）

实际上 腾讯云 香港云  内网机器多台   都通过腾讯云搭建zerotirt根服务打通udp 

腾讯云nginx转发的好处 ： 随便一台电脑都可以直接开始开发tvpage 
腾讯云nginx转发的坏处 ： 多了一层nginx 转发带来延迟  nginx本身的转发带来一些 其他的问题
